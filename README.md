LearnMate AI
An intelligent learning assistant that helps students master academic subjects through AI-powered personalized support. Users can ask questions, generate quizzes, summarize notes, and receive study recommendations, with the system offering:
✅ 💬 Academic Chatbot – Instant doubt clearing with contextual, step-by-step explanations
✅ 📝 Custom Quiz Generator – Multiple-choice quizzes on any topic with adjustable difficulty (Easy/Medium/Hard) and 3-10 questions
✅ 📄 Smart Notes Summarizer – Condense lengthy study notes into concise summaries (bullet points to detailed breakdown)
✅ 🎯 Personalized Study Recommendations – Custom learning plans based on learning style (Visual/Auditory/Reading/Kinesthetic), time availability, and goals
✅ 📊 Instant Grading – Quiz submission with auto-scoring and detailed explanations for each answer
✅ 💾 Persistent Chat History – Maintains conversation context for better learning continuity
Built for SDG 4 - Quality Education: Bridging the personalized learning gap for students in under-resourced schools and remote areas with no access to private tutors or mentoring.
Tech Stack: Streamlit | Python | Groq API (LLaMA 3.3 70B) | Hugging Face Spaces | python-dotenv
